What is morse code?


Morse code was named after Samuel Morse and other inventors who helped establish this language in the early 19th century. American and international morse code exist independently where the later is more accurate and easier to learn with short dashes.  This website presents the international Morse code. The International Morse code encodes the 26 basic Latin letters A through Z. Upper and lower case letters are not distinguished. Morse code symbols are formed by dits and dahs. The short bursts of sound or light are called “dit”, and the long bursts are called “dah”. Visually, they are written . are dits _ are dahs The Morse system was first based on the English alphabet, but did not work for other languages. In 1848, Fredrich Clemens Gerke refined the Morse Code into the Hamburg Alphabet. It's this alphabet that we now know as the International Morse Code.
